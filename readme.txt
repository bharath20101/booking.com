Steps to build and run test:
1) Clone the solution to local
2) Open pom.xml and save the file for all the dependencies to be configured
3) right click on pom.xml and click RunAs-> Maven test

Note:
The test will be run in chrome browser by default, ensure the driver version configured and browser version installed is compatible
To change browser updated browser name in config.properties

Tools used:
Selenium, TestNG, Cucumber, Extent Report, Maven.
